# supply-chain-claw

A long-running Next.js project to build a global, explorable map of supply chains.

## Vision
Track how goods move across:
- raw materials
- components (CPU/GPU and beyond)
- manufacturing
- logistics
- retail

## Current phase
- Base Next.js app scaffold
- Project roadmap
- Map placeholder UI

## Next milestones
1. Add a real world map with country overlays.
2. Add CPU/GPU supply chain data model + first visualization.
3. Add raw materials and food datasets.
4. Add source citations and confidence scoring.
